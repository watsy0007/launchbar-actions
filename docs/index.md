## 安装

1. 安装LaunchBar6
2. 下载列表中的某项action
3. 解压以后打开
4. 弹出安装的对话框中点击 安装 按钮

## actions

| action | 说明 |
| ------ | ------ |
| [public ip][pip] | 获取外网IP |
| [有道翻译][ydfy] | 通过有道api 翻译单词 |
| [kill process][kp] | 根据进程名杀死进程 |
| [BT search][bt] | 磁链搜索 |
| [tv show][tvshow] | 找美剧 |
